# Repo Map

- `frontend/pages/`: user-facing pages and API routes for the frontend app
- `frontend/components/`: shared UI, navbar, layout, charts, chatbot, shadcn-style primitives
- `backend/src/routes/`: API routing
- `backend/src/controllers/`: route handlers
- `backend/src/entities/`: TypeORM entities
- `backend/src/services/`: backend service logic
- `k8s/`: Kubernetes manifests and production overlay
- `terraform/`: AWS-centric infrastructure and optional HashiCorp modules
- `scripts/`: deploy, preflight, release, migration, and ops scripts
- `README.md`, `ARCHITECTURE.md`, `docs/DEPLOYMENT.md`, `docs/DEVOPS_GUIDE.md`: primary docs that often need sync
