# Repo Map

- `frontend/pages/`: user-facing pages and frontend API routes
- `frontend/components/`: shared UI, navbar, layout, charts, chatbot, and primitives
- `backend/src/routes/`: API routing
- `backend/src/controllers/`: route handlers
- `backend/src/entities/`: TypeORM entities
- `backend/src/services/`: backend service logic
- `k8s/`: Kubernetes manifests and production overlay
- `terraform/`: AWS-centric infrastructure and optional HashiCorp modules
- `scripts/`: deploy, preflight, migration, release, and ops scripts
- `README.md`, `ARCHITECTURE.md`, `docs/DEPLOYMENT.md`, `docs/DEVOPS_GUIDE.md`: docs most likely to need sync
